###  Overview
This is a professional Telegram Bot prototype developed during my 6th day of learning backend development. The goal was to create a functional lead generation tool for micro-businesses that can capture user requests and notify the administrator instantly.

###  Key Features
* Interactive Menu: Uses inline keyboards for a clean user interface.
* Lead Generation: Captures user messages and phone numbers.
* Admin Notifications: Automatically forwards new customer requests to the admin's private Telegram (Backend logic).
* State Management: Basic logic handling for "Services", "About Us", and "Booking" sections.

###  Tech Stack
* Language: JavaScript (Node.js)
* Framework: Telegraf.js
* Environment: Windows PowerShell / VS Code

###  How to Run
1. Clone the repository.
2. Run npm install to install dependencies.
3. Create a .env file or replace the token in index.js with your own from @BotFather.
4. Run node index.js
